<h2>Project 1: <b>Freqtrade Strategy &amp; Market Analysis</b></h2>

<p><b>Quantitative analysis of OHLCV data and large-scale strategy metadata.</b></p>

<p>
This project focuses on market behavior analysis and automated evaluation of algorithmic trading
strategies built for Freqtrade.
</p>

<h4>OHLCV Analysis (WIP)</h4>
<ul>
  <li>Price action and volume trend analysis</li>
  <li>Indicator-driven market insights</li>
  <li>Statistical summaries and visualizations</li>
</ul>

<h4>Strategy Analysis (WIP)</h4>
<ul>
  <li>Automated processing of public strategies from <a href="https://strat.ninja">Strat.Ninja</a></li>
  <li>Extraction of structural, indicator, and performance metadata</li>
  <li>Generation of a unified strategy dataset for research</li>
</ul>

<b>Extracted Strategy Metadata</b>
<ul>
  <li>Trading mode (Spot / Futures)</li>
  <li>Timeframe</li>
  <li>DCA usage</li>
  <li>Long/Short capability</li>
  <li>Leverage</li>
  <li>Stoploss configuration</li>
  <li>Bias detection (e.g., lookahead bias)</li>
  <li>Profit metrics</li>
  <li>Indicator usage (binary feature columns)</li>
</ul>

<b>Output</b>
<ul>
  <li><code>strategies_metadata.ndjson</code></li>
  <li>One row per strategy, enabling:
    <ul>
      <li>Correlation analysis</li>
      <li>Feature importance studies</li>
      <li>Strategy filtering &amp; comparison</li>
    </ul>
  </li>
</ul>

<b>Use Cases</b>
<ul>
  <li>Strategy benchmarking</li>
  <li>Indicator effectiveness research</li>
  <li>Quantitative trading research</li>
</ul>

<h2>Project 2: <b>ccxt-opensearch</b></h2>

<p><b>Market data ingestion, storage, and analytics pipeline using CCXT and OpenSearch.</b></p>

<p>
This project fetches real-time and historical cryptocurrency market data from multiple exchanges via
<b>CCXT</b> and ingests it into <b>OpenSearch</b> for scalable storage, querying, and visualization.
</p>

<b>Key Features</b>
<ul>
  <li>Automated OHLCV data collection (cron-ready)</li>
  <li>Structured time-based indices per exchange, symbol, and date</li>
  <li>OpenSearch Dashboards for exploration and analytics</li>
  <li>Dockerized stack for easy deployment</li>
  <li>Planned integrations:
    <ul>
      <li>TA-Lib indicators</li>
      <li>Indicator-based alerts</li>
      <li>Custom dashboards</li>
    </ul>
  </li>
</ul>

<b>Tech Stack</b>
<ul>
  <li>Python</li>
  <li>CCXT</li>
  <li>OpenSearch &amp; OpenSearch Dashboards</li>
  <li>Docker / Docker Compose</li>
</ul>

<b>Use Cases</b>
<ul>
  <li>Market data warehousing</li>
  <li>Indicator backtesting</li>
  <li>Alerting &amp; monitoring</li>
  <li>Multi-asset analytics at scale</li>
</ul>
